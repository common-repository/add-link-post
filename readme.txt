A plugin that allows you to post any article or news or the likes from any webpage to your blog simply by specifying the URL of that webpage from a widget. 

== Description ==
You can post any news or articles as your new blog post simply by specify its URL. This plugin will publish it as your new post with the title of that webpage as the post title and the excerpts of the webpage content along with its URL as the post content. Visit <a href="http://dev.coziplace.com/free-wordpress-plugins/add-link-post">Official</a> page. 

== Installation ==
1. Upload zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widgets in the wordpress admin dashboard and activate add-link-post widget.
4. Go to the front page of your site with administrative log-on, you will see the add-link-post widget. Put the URL of the link you wish to post on the text input of that widget and click publish.
